Search

Search Engines¶

OpenAIRE Explore - European open science data provider search

OSF Share - search engine for open scholarly content

Structured Data Search¶

Schema.org - structured data markup schemas for web content

Google Data Commons Sustainability - environmental and sustainability data platform

DataCommons.org - Google's platform unifying public datasets

Linked Open Data Cloud - visualization of interconnected open datasets

Scientific Knowledge Graphs¶

AIDA Knowledge Graph - 21M publications and 8M patents with CS topics

Open Knowledge Graph - Open University's linked open data platform

Open Research Knowledge Graph (ORKG) - structured scholarly knowledge with semantic descriptions

Amazon Science Knowledge Graphs - research publications on knowledge graph technology

Literature & Document Search¶

Open Bibliographic Databases¶

OpenAlex - free and open catalog of the global research system with 250M+ works, successor to Microsoft Academic Graph

The Lens - free, open scholarly search and analytics platform with patent data integration

Dimensions - research intelligence platform linking publications, grants, patents, clinical trials (free basic access)

General Search Engines¶

Google Scholar - based on Google's powerful knowledge graph

Bielefeld Academic Search Engine (BASE) - provides more than 240 million documents from more than 8,000 content providers

BioOne - aggregator of bioscience research journals

CERN Document Server - CERN's repository for physics publications

CiteSeerX - digital library for computer science literature

CORE - 207M+ open access articles from 10,000+ providers

DeepDive - Stanford's knowledge extraction system

  • PaleoDeepDive - paleobiology literature mining

  • GeoDeepDive (xDD) - geoscience digital library and knowledge extraction

Directory of Open Access Journals (DOAJ) - 20,000+ peer-reviewed open access journals

Mendeley - reference manager and academic social network

Microsoft Academic - Microsoft's academic search engine (discontinued 2021)

Organic ePrints - open access archive for organic agriculture

PubMed - biomedical literature database with 35M+ citations

ResearchGate - academic social network for sharing papers

Science.gov - US government science information portal

ScienceDirect - Elsevier's full-text scientific database

ScienceOpen - open research and publishing network

Scopus - Elsevier's citation database covering 27,000+ journals

Semantic Scholar - AI-powered research literature search

SSRN - Social Science Research Network preprint repository

UpToDate - evidence-based clinical decision support

Preprint Servers¶

arXiv - pioneering preprint server for physics, mathematics, computer science, quantitative biology, and statistics (since 1991)

bioRxiv - preprint server for biology operated by openRxiv nonprofit (founded 2013)

medRxiv - preprint server for health sciences operated by openRxiv nonprofit (founded 2019)

ChemRxiv - preprint server for chemistry hosted by American Chemical Society

EarthArXiv - preprint service for Earth science research

PsyArXiv - preprint archive for psychological science

SocArXiv - open archive of social science research

engrXiv - preprint server for engineering

AgriXiv - preprint service for agriculture and allied sciences

RePEc - Research Papers in Economics, collaborative effort of hundreds of volunteers

Query Languages¶

Web Ontology Language (OWL) - W3C semantic web ontology language

Resource Description Framework (RDF) - W3C standard for representing linked data

SPARQL - query language for RDF databases

GeoSPARQL - geographic extension to SPARQL query language
